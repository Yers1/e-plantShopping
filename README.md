# Paradise Nursery Shopping Cart

This is the final project for the Front-End Development course. 
It is a shopping cart application for an online plant shop called "Paradise Nursery".

## Features
- Landing page with company name and "Get Started" button
- Product listing page with plants grouped into categories
- Shopping cart page with dynamic updates of item quantity and total cost
- Add to Cart, Increase, Decrease, and Delete buttons
- Responsive navigation bar
